# Installation

## From crates.io

```bash
cargo install linewise
```

## From Source

```bash
git clone https://github.com/monokrome/linewise
cd linewise
cargo install --path .
```

## Pre-built Binaries

Download pre-built binaries from the [GitHub Releases](https://github.com/monokrome/linewise/releases) page:

- `lw-linux-x86_64` - Linux x86_64
- `lw-linux-aarch64` - Linux ARM64
- `lw-macos-x86_64` - macOS Intel
- `lw-macos-aarch64` - macOS Apple Silicon
- `lw-windows-x86_64.exe` - Windows x86_64

## Verify Installation

```bash
lw --version
```
